# 3D Крестики нолики

Итоговый экзаменационный проект по дисциплине "Веб-разработка". Мультиплеерная игра в крестики-нолики с 3D графикой, системой аккаунтов и сохранением прогресса.

## Ссылка на игру
[https://tictactor-16xl.onrender.com](https://tictactor-16xl.onrender.com)

## Описание
Игроки регистрируются, создают комнаты и играют друг против друга в реальном времени. Каждый игрок видит аватар противника в 3D пространстве. После завершения матча статистика побед и поражений автоматически сохраняется в базу данных MongoDB.

### Игровые механики
- Управление: Q/Й — захват мыши для вращения камеры, колёсико — зум, ЛКМ — ход.
- Вид от первого лица (камера внутри аватара игрока).
- Аватар противника поворачивается в сторону клетки при ходе.
- Кастомизация: выбор формы (сфера/куб) и цвета при регистрации.

## Технологии
- Frontend: HTML5, CSS3, JavaScript, Three.js
- Backend: Node.js, Express
- Real-time: Socket.io
- Database: MongoDB Atlas (Mongoose)
- Auth: JWT, bcrypt
- Deploy: Render.com

## Установка и запуск локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ТВОЙ_НИК/tictactoe3d.git
   cd tictactoe3d
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Создайте файл `.env` в корне проекта:
   ```env
   PORT=3000
   MONGO_URI=mongodb+srv://user:pass@cluster...
   JWT_SECRET=mysecretkey123
   ```

4. Запустите сервер:
   ```bash
   npm run dev
   ```

5. Откройте в браузере: http://localhost:3000/auth.html

## Структура проекта
- `server/` — бэкенд (API, сокеты, модели БД)
- `client/` — фронтенд (3D сцена, UI авторизации и лобби)
- `client/src/game.js` — основная логика игры на Three.js
